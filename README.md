# AWS Terraform module for creating two web servers in HA mode and an ELB

## Requirements

| Name | Version |
|------|---------|
| terraform | >= 0.12.7, < 0.14 |
| aws | >= 2.68, < 4.0 |


## Usage

To run this example you need to cd into environment/web directory and execute:

```bash
$ terraform init 
$ terraform plan 
$ terraform apply
```
